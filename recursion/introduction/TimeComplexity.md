# Time Complexity

## Sorting Algorithms

| Algorithm         | Best Case | Average Case | Worst Case | Space Complexity |
|------------------|-----------|-------------|------------|------------------|
| **Bubble Sort**  | Ω(n)      | Θ(n²)       | O(n²)      | O(1)             |
| **Selection Sort** | Ω(n²)   | Θ(n²)       | O(n²)      | O(1)             |
| **Insertion Sort** | Ω(n²)   | Θ(n²)       | O(n²)      | O(1)             |
| **Selection Sort** | Ω(n²)   | Θ(n²)       | O(n²)      | O(1)             |
