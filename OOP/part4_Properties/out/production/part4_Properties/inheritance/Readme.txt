super keyword - whenever a subclass needs to refer to the super class from which it is defined, the super keyword is used.

super for any class  refers to the class which is directly above it.

* every class's ultimate super class is the object class of java.


uses of super keyword -

1- to call the parent class constructor.

2- can be used as 'this' keyword. we can write super.variableName if we explicitly want to refer to the super class
